# Sales Performance Analysis

## Overview
This project analyzes simulated sales data to uncover revenue trends across regions and products.

## What I Did
- Created a dataset representing sales transactions
- Analyzed revenue by region and product
- Visualized key performance metrics using bar charts

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Revenue varies across regions
- Certain products generate higher sales
- Data visualization helps identify business trends
